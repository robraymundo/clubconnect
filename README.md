# Club Membership Management System

## Project Overview
The Club Membership Management System is designed to simplify and automate the management of clubs, members, memberships, and payments.  
It provides a centralized platform for administrators to manage clubs, approve membership requests, and track payments, while allowing members to apply for memberships, monitor application status, and view payment history.

---

## Features
- Role-based login for Admin and Members  
- Admin dashboard to manage clubs, members, and payments  
- Members can view available clubs, apply for memberships, and track application status  
- Track membership status (Active/Inactive)  
- Payment requests handling and verification  

## Folder Structure
- `SourceCode/` - C# project files
- `Documentation/` - Reports, diagrams, screenshots

## Installation
1. Open the `SourceCode/ClubMembership/ClubMembership.sln` file in Visual Studio.
2. Restore NuGet packages if needed.
3. Set up the MySQL database using `Database/ClubMembershipDB.sql`.

## Usage
1. Run the project in Visual Studio.
2. Login using your credentials.
3. Admin and User dashboards will display based on role.
